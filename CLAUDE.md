# Seemless Design

## Frontend Rule
When doing ANY frontend/UI work in this project, always follow the
frontend-design skill at `.claude/skills/frontend-design/SKILL.md`.
Read it before writing any CSS, styling, or component code.

## Project
A minimal writing tool with inline question resolution via Groq.
- Stack: Next.js (App Router), Tailwind, Tiptap, Groq SDK
- One page, one API route, no complexity
